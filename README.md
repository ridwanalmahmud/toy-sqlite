### Prerequisite
```
pacman -S make gcc
```

### Makefile

- Make the executable
```
make
```
- Make and Run the executable
```
make run
```
- Remove the executables
```
make clean
```
- Build the tests
```
make build-test
```
- Run the tests
```
make run-test
```
- Build and run the tests
```
make test
```
